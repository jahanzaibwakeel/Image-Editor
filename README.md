# Canvas Drawing Application

## Overview
This application is a feature-rich canvas drawing tool that enables users to draw, shape, and add text with customizable colors and brush sizes. Built using React and Zustand for state management, it provides an intuitive user interface for creative expression.

## Features
- **Drawing Tools**: Brush, Eraser, Shapes (Rectangle, Circle, Line)
- **Text Input**: Add customizable text to the canvas
- **Color Picker**: Choose colors for drawing and shapes
- **Opacity Control**: Adjust brush opacity for varied effects
- **Undo/Redo Functionality**: Manage drawing history
- **Responsive Design**: Adaptive layout for different screen sizes
